# AI Matching Module - Deterministic Engine

A robust, mathematically sound candidate-to-job matching engine built as a deterministic baseline. This system evaluates candidates against job requirements using a weighted scoring model, exponential penalty curves, and strict must-have multipliers to prevent "false positive" matches.

## 🏗️ Architecture & Features

- **Strict Type Validation:** Uses `Pydantic` to ingest, validate, and sanitize JSON data, ensuring the scoring engine never crashes due to malformed inputs.
- **O(1) Skill Normalization:** An inverted dictionary approach aliases raw skills (e.g., `js`, `react.js`) into standardized keys instantly.
- **Explainable AI:** Every calculation generates an explicit, human-readable audit trail explaining exactly why a candidate received their score and dynamically summarizes their fit.
- **Environment Driven:** Weights and mathematical caps are separated from the core logic into a `.env` file via `pydantic-settings`.
